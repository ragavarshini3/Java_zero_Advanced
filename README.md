# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/Java-Zero-to-Advanced.git
```

### Navigate to the Repository

```bash
cd Java-Zero-to-Advanced
```

### Compile a Java Program

```bash
javac FileName.java
```

### Run a Java Program

```bash
java FileName
```
