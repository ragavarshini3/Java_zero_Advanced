# 🛠 Software Tools

Before writing Java programs, we need a few essential tools. The tools may vary slightly depending on the operating system.

## 🪟 For Windows

* **Java JDK (Oracle JDK / OpenJDK)** – Install Java and configure `JAVA_HOME`
* **IntelliJ IDEA / VS Code** – Popular IDEs for Java development
* **Command Prompt / PowerShell** – Compile and run Java programs
* **Git Bash** – Useful for Git and GitHub operations

---

## 🍎 For macOS

* **Java JDK (OpenJDK / Oracle JDK)** – Installed using Homebrew or manual setup
* **IntelliJ IDEA / VS Code** – Optimized Java development environments
* **Terminal (Zsh/Bash)** – Run Java commands and manage projects
* **Homebrew** – Package manager for installing Java and development tools

---

## 🚀 For Apple Silicon (M1 / M2 / M3 Macs)

* **ARM-Compatible JDK** (OpenJDK, Azul Zulu, Oracle JDK)
* **IntelliJ IDEA (Apple Silicon Edition)** – Better performance and lower power consumption
* **VS Code (ARM Version)** – Lightweight and fast
* **Terminal + Rosetta 2 (Optional)** – Compatibility for older applications

---

## 🔧 Common Tools (All Platforms)

* **Git & GitHub** – Version control and project hosting
* **Debugger Tools** – Find and fix errors efficiently
* **Web Browser** – Access documentation and learning resources
* **Java Documentation** – Official reference for Java APIs and features

---

## ✅ Conclusion

Choosing the right tools for your operating system ensures a smooth Java development experience. Once the tools are installed and configured, you are ready to start writing and running Java programs.
