# What is Java?

Java is a high-level, object-oriented, and platform-independent programming language developed by Sun Microsystems in 1995. It was created by James Gosling and later became one of the most popular programming languages in the world.

Java follows the principle:

> **"Write Once, Run Anywhere (WORA)"** 
 
This means a Java program can run on any device that has a Java Virtual Machine (JVM) installed.
 
---

## Why Java? 

Java is widely used because it is:

* Simple and easy to learn
* Object-Oriented
* Platform Independent
* Secure
* Robust and Reliable
* Multithreaded
* High Performance
* Portable

---

## Where is Java Used?

Java is used in many areas of software development:

### Web Applications

Popular frameworks like Spring Boot are built using Java.

### Mobile Applications

Android applications are primarily developed using Java and Kotlin.

### Desktop Applications

Software such as IDEs and management systems can be built with Java.

### Enterprise Applications

Large-scale business applications often use Java because of its reliability.

### Cloud and Distributed Systems

Many cloud-based services use Java for backend development.

---

## How Java Works?

Java programs follow these steps:

1. Write Java source code (`.java` file)
2. Compile using the Java Compiler (`javac`)
3. Generate Bytecode (`.class` file)
4. JVM executes the bytecode

### Flow

```text
Java Source Code (.java)
          ↓
       Compiler
          ↓
 Bytecode (.class)
          ↓
          JVM
          ↓
       Output
```
