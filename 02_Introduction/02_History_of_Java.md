 # History of Java

Java was developed by James Gosling and his team at Sun Microsystems in 1991.

The project was initially called "Oak" because of an oak tree outside Gosling's office. Later, the name was changed to "Java" in 1995.

Java was designed to be platform-independent, secure, and easy to use.

---

## Timeline

### 1991
Project Oak started by James Gosling.

### 1995
Oak renamed to Java and officially released.

### 1996
Java 1.0 released.

### 2006
Java became open-source through OpenJDK.

### 2010
Oracle acquired Sun Microsystems.

### Present
Java is widely used in web, mobile, enterprise, and cloud applications.

---

## Creator of Java

James Gosling is known as the Father of Java.

---

## Conclusion

Java has evolved into one of the most popular programming languages in the world.
